# Dynamic Ads Content

Ein Full-Stack-Projekt mit FastAPI Backend und React Frontend.

## Struktur

- `backend/` - FastAPI Backend-API
- `frontend/` - React Frontend mit Vite

## Schnellstart

### Backend

```bash
cd backend
python -m venv venv
.\venv\Scripts\activate  # Windows
pip install -r requirements.txt
cp .env.example .env
python main.py
```

Backend läuft auf: http://localhost:8000

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend läuft auf: http://localhost:5173

## Entwicklung

- Backend API-Dokumentation: http://localhost:8000/docs
- Frontend Development: http://localhost:5173
